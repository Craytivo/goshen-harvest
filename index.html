Preventative Maintenance Services Provided

Our preventative maintenance service includes a full checklist for the most-used machines in your gym. We keep them running smoothly so you can focus on your members.

Bikes

  • • Test ride.
  • • Check: Pedals/crank arms, display, keypad, and chain tightness.
  • • Lubricate if needed.
  • • Safety check includes all exterior bolts.

Treadmills

  • • Test ride.
  • • Check: Belt/deck wear, wax level, power, alignment, display panel, and safety.
  • • Remove motor cover and clean out dust.
  • • Calibrate if needed.

Ellipticals

  • • Test ride.
  • • Check: Pedals/crank arms, keypad, and chain tightness.
  • • Lubricate if needed.
  • • Safety check includes exterior bolts.

Steppers

  • • Test ride for rust/bubbling.
  • • Inspect: Belts/chains and adjust.
  • • Lubricate chains and degrease.
  • • Check: Pedals, springs, and machine overall.

Rowers

  • • Clean monorail.
  • • Check: Pedals/crank arms, display, keypad, and chain.
  • • Lubricate if needed.
  • • Safety check on bolts and connections.

Weight Machines

  • • Test ride.
  • • Check and lubricate the chain.
  • • Adjust: Bottom seat rollers.
  • • Inspect: Shock cords, handle monitors, and function.
